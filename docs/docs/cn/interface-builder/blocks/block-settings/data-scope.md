# 设置数据范围

## 介绍

设置数据范围是指为数据区块定义默认筛选条件，用户可根据业务需求灵活调整数据范围，但无论进行何种筛选操作，系统都会自动应用该默认筛选条件，确保数据始终符合指定范围的限制。

### 操作符

不同类型的字段支持不同的操作符，例如文本字段支持等于、不等于、包含等操作符，数字字段支持大于、小于等操作符，日期字段则支持在范围内、在特定日期之前等操作符。
