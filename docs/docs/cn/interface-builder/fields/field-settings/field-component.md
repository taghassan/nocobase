# 字段组件

## 介绍

字段组件是用于展示字段内容的载体。
部分字段内置多种展示组件，用户可根据实际需求选择合适的组件，以不同的方式呈现字段的值

示例: 阅读态 URL 字段可以切换为预览组件、URL组件（默认）
