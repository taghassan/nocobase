# 子详情

## 介绍

子详情是子表单在阅读模式下的对应组件，与标签和标题组件相比，子详情不仅可以展示更多本表数据，还可以配置展示关系表数据信息，多层级的关系数据以嵌套详情的形式清晰展示。
